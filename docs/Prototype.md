# SyncoBoard 高保真原型图说明

> 说明：本文件以“高保真原型说明书”的形式描述关键页面布局、视觉规范与组件状态，便于后续在 Figma/Sketch 中快速落地。待设计稿产出后，可将图片链接替换为真实原型图。

## 1. 视觉规范
- 基础色板
  - 主色：#3B82F6
  - 成功：#22C55E
  - 警告：#F59E0B
  - 危险：#EF4444
  - 中性色：#111827 / #6B7280 / #E5E7EB / #F9FAFB
- 字体
  - 标题：18-24px，Semibold
  - 正文：14-16px
- 圆角与阴影
  - 卡片圆角 8px
  - 浮层阴影：0 4px 12px rgba(0,0,0,0.08)
- 间距系统：8px 基准（8/16/24/32）
- 图标风格：线性图标，2px 线宽

## 2. 页面原型说明
### 2.1 登录页 `/login`
- 结构：左侧品牌区 + 右侧登录表单
- 背景：渐变或插画背景
- 表单元素：邮箱、密码、登录按钮、注册链接
- 状态：
  - 默认/聚焦/错误（红色提示）

### 2.2 总览 `/app/dashboard`
- 顶部：Header（品牌 + 用户菜单）
- 左侧：Sidebar（项目导航）
- 主内容：
  - KPI 卡片 3 个（总卡片、逾期、活跃成员）
  - 折线图：近 7 天新增卡片
  - 饼图/柱状图：卡片状态分布

### 2.3 项目列表 `/app/projects`
- 顶部：搜索框 + 筛选
- 列表：卡片式项目卡（名称、描述、成员数、创建时间）
- 空状态：插画 + “创建项目”按钮

### 2.4 看板 `/app/projects/:id/board`
- 顶部：项目名称 + 成员头像 + 邀请按钮
- 看板列：横向滚动
- 卡片样式：
  - 标题 + 标签 + 优先级图标 + 截止日期
  - 拖拽时卡片放大并显示阴影
- 交互：拖拽落位时显示占位阴影
- 空列提示：展示“拖拽卡片到这里”文案

### 2.5 聊天 `/app/projects/:id/chat`
- 左侧：成员列表（可折叠）
- 右侧：消息列表 + 输入框
- 消息气泡：区分自己与他人
- 未读提示：红点或数值 badge
- 系统消息：如“成员加入”以灰色细文本展示

### 2.6 活动流 `/app/projects/:id/activity`
- 时间线样式：图标 + 文本
- 支持筛选（类型/时间范围）

### 2.7 管理员用户管理 `/app/admin/users`
- 表格：用户信息 + 状态 + 操作（禁用/启用）
- 搜索与筛选

## 3. 组件规格
- 项目卡片：高度 120px，包含标题、描述摘要与统计
- 看板列宽：280px，列间距 16px
- 卡片高度：自适应，最小 80px
- 按钮：主按钮 36px 高度，圆角 6px
- 输入框：高度 36px，圆角 6px，聚焦态描边主色

## 4. 交互动效
- 页面切换：淡入 200ms
- 拖拽：shadow 动效 + 位置占位
- 图表加载：骨架屏 600ms
- 弹窗：缩放进入 150ms
