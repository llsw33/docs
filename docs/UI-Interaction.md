# SyncoBoard UI 交互说明书

## 1. 全局交互规范
- Header 始终显示用户头像与快捷菜单
- Sidebar 支持展开/折叠，状态本地缓存
- 所有列表页支持搜索、筛选与空状态引导

## 2. 登录与注册
- 登录按钮 disabled 直到输入合法
- 错误提示在输入框下方显示
- 登录成功后自动跳转 `/app/dashboard`

## 3. 项目列表
- 点击项目卡进入看板
- 新建项目按钮固定在列表顶部
- 空状态显示插画与 CTA

## 4. 看板交互
- 卡片拖拽：
  - 拖拽开始：卡片放大、阴影增强
  - 拖拽中：显示占位区域
  - 拖拽结束：乐观更新 + 失败回滚提示
- 卡片点击：右侧弹窗显示详情
- 标签与优先级可在卡片详情编辑

## 5. 聊天交互
- Enter 发送、Shift+Enter 换行
- 新消息到达时自动滚动至底部（若用户在底部）
- 未读提示：顶部展示未读条，点击跳转最新

## 6. Dashboard 交互
- 图表 hover 显示 tooltip
- KPI 卡片点击进入筛选结果（可选）

## 7. 活动流交互
- 时间线支持按类型筛选
- 点击活动可跳转至关联卡片

## 8. 用户管理交互
- 搜索用户即时过滤
- 禁用用户需二次确认

## 9. 异常与空状态
- 网络异常：toast 提示 + 重试按钮
- 权限不足：弹窗提示并返回项目列表
