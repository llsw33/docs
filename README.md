本仓库用于集中维护 SyncoBoard 项目的完整文档体系，覆盖从商业需求到详细设计、数据库与接口规范、以及高保真原型与交互说明。

## 适用范围
- MVP 目标：看板实时同步 + 项目聊天实时同步
- 技术栈：Spring Boot 3 + Spring Security + JWT + WebSocket(STOMP)；Vue3 + Element Plus + ECharts
- 数据存储：PostgreSQL + Redis
